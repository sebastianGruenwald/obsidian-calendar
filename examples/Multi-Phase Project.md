# Multi-Phase Project

This project has multiple important dates across different months.

## Phase Dates
- **February 14**: Phase 1 Kickoff
- **March 14**: Phase 2 Review
- **April 14**: Final Delivery

## Adding More Dates
You can add more dates to this event using the right-click context menu:
1. Right-click on the event in the calendar
2. Select "Add date to event"
3. Choose a date from the picker
4. The new date will be added to the frontmatter automatically

This is useful for:
- Adding ad-hoc meeting dates
- Tracking multiple milestones
- Scheduling follow-ups
